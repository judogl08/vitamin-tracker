Vitamin Tracker Pro v7

Добавлено в веб-версию:
- тёмная, светлая и системная темы;
- отдельные цвета мужчины и женщины с возможностью изменить акцент;
- локальная блокировка через Face ID / Touch ID / код устройства (WebAuthn/passkey);
- полноценные курсы с датой начала, длительностью и паузой;
- журнал пропусков за 30 дней;
- учёт остатков в банке и автоматическое списание после отметки приёма;
- порог «купить», список покупок и календарное напоминание;
- редактирование времени каждого витамина;
- добавление, редактирование и удаление добавок;
- резервная копия всех данных в JSON с сохранением через «Файлы → iCloud Drive»;
- восстановление из резервной копии;
- макет будущего iPhone-виджета;
- обновлённый интерфейс с нижней навигацией.

Ограничения веб-версии:
- настоящий iPhone Home Screen Widget требует нативного приложения и WidgetKit;
- прямой доступ к Apple Health требует нативного приложения, HealthKit capability и разрешения пользователя;
- автоматическая iCloud-синхронизация требует CloudKit-контейнера и Apple Developer;
- резервная копия в iCloud Drive в веб-версии выполняется через меню «Поделиться / Сохранить в Файлы».

Обновление GitHub Pages:
1. Загрузить все файлы из папки в корень репозитория с заменой.
2. Дождаться зелёной галочки GitHub Pages.
3. Открыть сайт с ?v=7.
4. На iPhone удалить старый ярлык и добавить сайт на экран Домой заново.
