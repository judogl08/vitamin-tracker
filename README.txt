Vitamin Tracker v4

Изменения:
- Установлена выбранная тёмно-синяя иконка с оранжевой капсулой и галочкой.
- Подготовлены отдельные размеры для iPhone и PWA: 180, 192 и 512 px.
- Добавлен favicon 32 px.
- Иконки получили новые имена с суффиксом v4, чтобы браузер не использовал старый кэш.
- Кэш приложения обновлён до vitamins-v4.
- Сохранены функции версии v3: добавление, редактирование и удаление витаминов,
  настройка времени, отдельные профили мужчины и женщины, календарь и контроль выполнения.

Важно:
При замене файлов на том же адресе GitHub Pages данные localStorage обычно сохраняются,
поскольку адрес сайта не меняется. Для смены иконки на экране iPhone может потребоваться
удалить старый ярлык и заново выполнить Safari → Поделиться → На экран Домой.
